# BillAnalyzAgent

基于 Spring Boot 的账单分析后端项目骨架。

## 环境要求

- Java 17+
- Maven Wrapper（项目已自带，无需单独安装 Maven）

## 启动

```bash
./mvnw spring-boot:run
```

应用默认运行在 `http://localhost:8080`。

## 接口

- `GET /api/v1/health`：应用健康检查
- `GET /actuator/health`：Spring Boot Actuator 健康检查

## 测试

```bash
./mvnw test
```
